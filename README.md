# MealGenerator
Mealgenerator est une application développé en Python qui a pour objectif d'aider la planification des repas. Le concept est simple proposer des plats de saison, ainsi qu'une liste de course.

## Librairies
Manipulation du JSON => jmespath

## Compilation
Afin de rendre l'outil plus simple d'utilisation, il est possible de le compiler avec `pyinstaller`
```
pyinstaller -F app.py
```
